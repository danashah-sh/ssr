# SSR – Node.js, Express og Database

Dette prosjektet er en oppgave i Server Side Rendering.  
Jeg har laget en server med **Express** som kan sende både **HTML** og **JSON**, og hente data fra en **PostgreSQL-database**. Vi har også lagt inn statiske filer i en public-mappe.

---

## Teknologier

- Node.js  
- Express  
- PostgreSQL
